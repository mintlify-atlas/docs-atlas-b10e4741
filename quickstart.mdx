# Quick start

This guide walks you through creating a fully interactive Squiggly Slider in Svelte. You'll build a slider with two-way data binding and custom wave animations.

<Steps>
  <Step title="Import the component">
    Start by importing `SquigglySlider` into your Svelte component:

    ```svelte
    <script>
      import SquigglySlider from 'squiggly-sliders';
    </script>
    ```
  </Step>

  <Step title="Add the minimal slider">
    Create a basic slider with required props:

    ```svelte
    <script>
      import SquigglySlider from 'squiggly-sliders';
    </script>

    <SquigglySlider
      min={0}
      max={10}
      value={6}
      active={'--md-sys-color-primary'}
      passive={'--md-sys-color-on-surface'}
    />
    ```

    This renders an animated slider from 0 to 10 with the value set at 6. The `active` and `passive` props reference CSS variables that control the wave colors.
  </Step>

  <Step title="Bind the value">
    Use Svelte's `bind:value` directive to create two-way data binding:

    ```svelte
    <script>
      import SquigglySlider from 'squiggly-sliders';
      let value = 6;
    </script>

    <SquigglySlider
      bind:value
      min={0}
      max={10}
      active={'--md-sys-color-primary'}
      passive={'--md-sys-color-on-surface'}
    />

    <p>Current value: {value}</p>
    ```

    Now when you drag the slider, the `value` variable updates automatically, and you can also change `value` programmatically to update the slider.
  </Step>

  <Step title="Customize the animations">
    Add wave customization props to control the animation physics:

    ```svelte
    <script>
      import SquigglySlider from 'squiggly-sliders';
      let value = 6;
    </script>

    <SquigglySlider
      bind:value
      min={0}
      max={10}
      step={1}
      active={'--md-sys-color-primary'}
      passive={'--md-sys-color-on-surface'}
      activeAmplitude={4}
      passiveAmplitude={2}
      activeWavelength={1}
      passiveWavelength={1}
      speedFactor={4}
    />
    ```

    <Note>
      - `activeAmplitude` and `passiveAmplitude` control wave height (0-7)
      - `activeWavelength` and `passiveWavelength` control wave frequency (0.25-7)
      - `speedFactor` controls animation speed (1-8, where higher is faster)
    </Note>
  </Step>

  <Step title="Define your CSS variables">
    Ensure your CSS includes the color variables referenced in the `active` and `passive` props:

    ```css
    :root {
      --md-sys-color-primary: rgb(97 97 24);
      --md-sys-color-on-surface: rgb(28 28 20);
    }

    @media (prefers-color-scheme: dark) {
      :root {
        --md-sys-color-primary: rgb(189 189 106);
        --md-sys-color-on-surface: rgb(228 228 220);
      }
    }
    ```

    The slider automatically detects and responds to theme changes via the `prefers-color-scheme` media query.
  </Step>
</Steps>

## Complete example

Here's the full working example with all features:

```svelte
<script>
  import SquigglySlider from 'squiggly-sliders';
  let value = 6;
</script>

<div class="slider-container">
  <h2>Squiggly Slider Demo</h2>
  
  <SquigglySlider
    bind:value
    min={0}
    max={10}
    step={1}
    active={'--md-sys-color-primary'}
    passive={'--md-sys-color-on-surface'}
    activeAmplitude={4}
    passiveAmplitude={2}
    activeWavelength={1}
    passiveWavelength={1}
    speedFactor={4}
  />
  
  <p>Current value: {value}</p>
  
  <button on:click={() => value = Math.floor(Math.random() * 11)}>
    Random value
  </button>
</div>

<style>
  .slider-container {
    max-width: 500px;
    padding: 2rem;
  }
  
  h2 {
    margin-bottom: 1.5rem;
  }
  
  p {
    margin-top: 1rem;
    font-size: 1.125rem;
  }
  
  button {
    margin-top: 1rem;
    padding: 0.5rem 1rem;
    border-radius: 0.375rem;
    background: var(--md-sys-color-primary);
    color: white;
    border: none;
    cursor: pointer;
  }
</style>
```

## Understanding the wave physics

The wave animations use sine functions to create smooth, continuous motion:

- **Amplitude**: Controls the vertical displacement (wave height). Higher values create more dramatic waves.
- **Wavelength**: Controls the horizontal frequency. Lower values create tighter, more frequent waves.
- **Speed factor**: Multiplies the time variable in the animation loop. Higher values speed up the wave motion.

The active and passive waves can have independent physics, letting you create visual contrast between the filled and unfilled portions of the slider.
